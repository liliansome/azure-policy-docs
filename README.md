## Overview 
This is a comprehensive collection of Azure Policy templates, examples, and automation scripts for cloud governance, compliance, and infrastructure management.

##  Features
Multi-Format Templates: YAML, JSON, Bicep, ARM templates

Complete Policy Examples: Location restrictions, tagging, VM SKUs, security baselines

Automation Scripts: Deployment, compliance checks, remediation

CI/CD Integration: GitHub Actions, Azure DevOps pipelines

Documentation: Best practices, getting started guides, reference materials

Testing Environment: Docker containers for local validation
